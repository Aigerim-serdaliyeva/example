Window.location - возвращает объект Location с информацией о текущем расположении документа. location.hash - часть URL, которая идет после символа решетки '#', включая символ '#' (#route) stopPropagation - предотвращает stopPropagation события в цепочке событий. - Метод объекта event stopPropagation останавливает "всплытие" вызова события к родительским элементам. preventDefault - предотвращает действие по умолчанию, которое браузер выполняет для этого события.
